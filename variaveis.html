Introdução à Variáveis e Tipos de Dados em JavaScript


1. O que são Variáveis?


Em JavaScript, uma variável é um contêiner para armazenar dados. Ela pode conter diferentes tipos de informações, como números, strings (texto), booleanos (verdadeiro ou falso), entre outros.



2. Declarando Variáveis:


Você pode declarar uma variável usando a palavra-chave var, let, ou const:

  • VAR - Era a forma mais comum de declarar variáveis antes do ES6 (ECMAScript 2015).
  • LET - É introduzido no ES6 e é preferido para declarar variáveis mutáveis.
  • CONST - Também é introduzido no ES6 e é preferido para declarar variáveis que não mudarão de valor.



3. Tipos de Dados em JavaScript:


  • Número: Representa valores numéricos. Pode ser um número inteiro ou decimal.




  • String: Sequência de caracteres, usada para representar texto.




  • Booleano: Representa um valor lógico - verdadeiro (true) ou falso (false).




  • Array: Estrutura de dados que armazena uma coleção de elementos.




  • Undefined: Valor atribuído automaticamente a variáveis que não foram inicializadas.




Entender variáveis e tipos de dados é fundamental para programar em JavaScript. Com esses conceitos, você pode armazenar e manipular informações de maneira eficaz em seus programas. Pratique e experimente diferentes tipos de dados para melhorar suas habilidades de programação!
