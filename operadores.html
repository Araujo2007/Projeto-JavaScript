Explorando Operadores em JavaScript para Iniciantes


O que são Operadores?


Operadores são símbolos especiais que realizam operações em valores e variáveis. Em JavaScript, existem diferentes tipos de operadores, incluindo operadores aritméticos, de atribuição, de comparação lógicos



1. Operadores Aritméticos:


Os operadores aritméticos são usados para realizar operações matemáticas básicas.

  • Adição (+) - Adiciona dois valores.
  • Subtração (-) - Subtrai um valor de outro.
  • Multiplicação (*) - Multiplica dois valores.
  • Divisão (/) - Divide um valor pelo outro.



2. Operadores de Atribuição:


Os operadores de atribuição são usados para atribuir valores a variáveis.

  • Atribuição simples (=) - Atribui um valor a uma variável.
  • Atribuição de adição (+=) - Adiciona um valor a uma variável existente.
  • Atribuição de subtração (-=) - Subtrai um valor de uma variável existente.
  • Atribuição de multiplicação (*=) - Multiplica uma variável existente por um valor.
  • Atribuição de divisão (/=) - Divide uma variável existente por um valor.



3. Operadores de Comparação:


Os operadores de comparação são usados para comparar dois valores.

  • Igual (==) - Verifica se dois valores são iguais.
  • NÃO igual (!=) - Verifica se dois valores não são iguais.
  • Estritamente igual (===) - Verifica se dois valores são iguais e do mesmo tipo.
  • Estritamente NÃO igual (!==) - Verifica se dois valores não são iguais e/ou não do mesmo tipo.
  • Maior que (>), Menor que (<), Maior ou igual (>=), Menor ou igual (<=) - Usado para comparar valores numéricos.



4. Operadores Lógicos:


Os operadores lógicos são usados para combinar expressões condicionais.

  • E lógico (&&) - Retorna verdadeiro se ambas as expressões forem verdadeiras.
  • OU lógico (||) - Retorna verdadeiro se pelo menos uma das expressões for verdadeira.
  • Negação lógica (!) - Inverte o valor de verdade de uma expressão.



Os operadores são ferramentas essenciais em JavaScript para manipular valores e controlar o fluxo do seu código. Entender como usar diferentes tipos de operadores é fundamental para construir programas eficientes e funcionais. Pratique com eles para se familiarizar e experimente diferentes combinações para alcançar o resultado desejado!
